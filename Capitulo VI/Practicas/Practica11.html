6.11 ¿Qué aparece en el diálogo de alerta cuando se ejecuta cada una de las siguientes instrucciones de JavaScript? Asuma que x = 2 y y=3.


  1. window.alert( "x = " + x);

    2. R: x = 2

  2. window.alert( "El valor de x + x es " + ( x + x ) );

    3. R: El valor de x + x es 4

  3. window.alert( "x =" );

    4. R: x =

  4. window.alert( ( x + y ) + " = " + ( y + x ) );

    5. R: 5 = 5
