6.5 Escriba una instrucción (o comentario) para realizar cada una de las siguientes tareas:


  1. Indique que una secuencia de comandos calculará el producto de trés enteros [Sugerencia: use texto que ayude a documentar una secuencia de comandos].

    2. // Calcula el producto de tres enteros.

  2. Declare las variables x, y, z y resultado.

    3. var x, y, z, resultado;

  3. Declare las variables xVal, yVal y zVal.

    4. var xVal, yVal, zVal;

  4. Pida al usuario que introduzca el primer valor, lea el valor del usuario y almacénelo en la variable xVal.

    5. xVal = window.prompt( "Escriba el primer entero:", "0");

  5. Pida al usuario que introduzca el segundo valor, lea el valor del usuario y almacénelo en la variable yVal.

    6. yVal = window.prompt( "Escriba el segundo entero:", "0");

  6. Pida al usuario que introduzca el tercer valor, lea el valor del usuario y almacénelo en la variable zVal.

    7. zVal = window.prompt( "Escriba el tercer entero:", "0");

  7. Convierta la cadena xVal en un entero y almacene el resultado en la variabla x.

    8. x = parseInt( xVal );

  8. Convierta la cadena yVal en un entero y almacene el resultado en la variabla y.

    9. y = parseInt( yVal );

  9. Convierta la cadena zVal en un entero y almacene el resultado en la variabla z.

    10. z = parseInt( zVal );

  10. Calcule el producto de los tres enteros contenidos en las variables x, y y z, y asigne el resultado a la variable resultado.

    11. resultado = x * y * z;

  11. Escriba una línea de texto de HTML5 que contenga la cadena "El producto es" seguida del valor de la variable resultado.

    12. document.writeln( "<h1>El producto es " + resultado + "</h1>";)
