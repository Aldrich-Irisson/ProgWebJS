Ejercicios de autoevaluación

8.1 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué

  1. El caso default es requerido en la instrucción de selección switch. Falso: El caso default es opcional. Si no se necesita una acción predeterminada, entonces no hay necesidad de un caso default.
  2. La instrucción break es requerida en el último caso de una instrucción de selección switch. Falso. La instrucción break se utiliza para salir de switch. La instrucción break no se requiere para el último caso de switch.
  3. La expresión (x > y && a < b) es verdadera si x> y es verdadera, o si a < b es verdadera. Falso. Ambas expresiones relacionales deben ser verdaderas para que toda la expresión sea verdadera cuando se utilice el operador &&.
  4. Una expresión que contiene el operador || es verdadera si uno o ambos de sus operadores son verdaderos. Verdadero
