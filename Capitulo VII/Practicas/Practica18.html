7.18 Sin ejecutar la siguiente secuencia de comandos, determine lo que imprime:

<script>
        var fila = 10;
        var columna;
        while (fila >= 1) {
            columna = 1;
            document.writeln("<p>");
            while (columna <= 10) {
                document.write(fila % 2 == 1 ? "<" : ">");
                columna++
            }
            --fila
            document.writeln("</p>");
        }
    </script>

Respuesta: Crea por fila una secuencia de símbolos y dependiendo si es fila par o impar escribe el símbolo: < o >
