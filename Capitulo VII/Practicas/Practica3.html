7.3 Escriba instrucciones en JavaScript para realizar cada una de las siguientes tareas:

  1. Asignar la suma de x y y a z, e incrementar el valor de x en 1 después del cálculo. Usar sólo una instrucción.

    2. z = x++ + y;

  2. Evaluar si el valor de la variable cuenta es mayor que 10. De ser así, imprimir "Cuenta es mayor que 10".

    3. if (cuenta > 10)

    document.writeln("Cuenta es mayor que 10");

  3. Decrementar la variable x en 1, luego restarla a la variable total. Usar sólo una instrucción.

    4. total-= --x;

  4. Calcular el residuo después de dividir q entre divisor y asignar el resultado a q. Escriba esta instrucción de dos maneras distintas.

    5. q %= divisor;

    q = q%divisor;
