Ejercicios de autoevaluación


7.1 Complete las siguientes oraciones:

  1. Todas las secuencias de comandos pueden escribirse en términos de tres tipos de estructura de control: ___secuencia___, ___selección___ y ___repetición___.

  2. La instrucción de selección doble ___if...else___ se utiliza para ejecutar una acción cuando una condición es verdadera, y otra acción cuando esa condición es falsa.

  3. Al proceso de repetir un conjunto de instrucciones un número específico de veces se llama repetición ___controlada por contador (o definida)___.

  4. Cuando no se sabe de antemano cuántas veces se repetirá un conjunto de instrucciones, se puede usar un valor ___centinela___ (o un ___valor de señal___, ___valor de bandera___ o ___valor de prueba___) para terminar la repetición.
